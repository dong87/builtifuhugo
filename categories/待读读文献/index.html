N9--多任务高斯过程推断多性状和多点试验中的缺失数据

Posted on October 1, 2017

  1. Hori T, Montcho D, Agbangla C, Ebana K. Multi ‑ task Gaussian process for imputing missing data in multi ‑ trait and multi ‑ environment trials. Theor. Appl. Genet. Springer Berlin Heidelberg; 2016;

要点： + 主要利用性状间的相关信息进行推断。 + 作者构建的工作框架能够利用多性状多环境数据间的相关，也能结合MET（多点试验）数据集标记基因型数据。 + 比较了多任务高斯过程和迭代调控PCA2个方法，也考虑了不同缺失率下的推断。 + 高斯的准确率高于PCA,尤其在高缺失率的情况下。
