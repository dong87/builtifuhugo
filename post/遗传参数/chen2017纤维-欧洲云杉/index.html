抽象 生长性状和实木之间的不良遗传相关性，以及纤维性状是针叶树育种计划中的关注点。为了评价选择对生长和固体木材性质对纤维尺寸的影响，我们调查了不同木纤维性状的早期选择的遗传和效率，以及它们与茎直径，木材密度，弹性模量（MOE）的相关性，微纤丝角（MFA）的挪威云杉（云杉 L）。该研究基于1990年在瑞典南部建立的两个大型开放授粉后代试验，其中包括来自524个家庭的材料，包括5618棵树。从每棵树采样两个增量芯。对于在径向（RFW）和切向（TFW）方向，纤维壁厚度（FWT）和纤维粗糙度（FC）的纤维宽度，使用SilviScan对环3-15确定从树干到树皮的径向变化。确定环8-11的纤维长度（FL）。基于环8-11使用关节位点数据的遗传性对于所有纤维维度性状是中等至高（0.24-0.51）。基于茎横截面平均值的遗传度从0.34变化到0.48，并在环6-9处达到平台。在具有环15处的横截面平均值的特定年龄的RFW，TFW，FWT和FC横截面平均值的“年龄”遗传相关性在环4-7处达到0.9。我们的研究结果表明密度和MOE与FC和FWT中，高积极的遗传相关性，中等和负RFW，低TFW和FL。几种选择方案的比较表明，当直径和MOE被共同考虑时达到最高的获利能力，在这种情况下，对任何纤维尺寸的影响是可忽略的。早期选择对于RFW的环5和对于TFW，FWT和FC的环8-10是高效的。

介绍 在挪威云杉（欧洲云杉（L.）H.岩溶。），茎的生长，直线度和分支角度通常评定为基于其经济价值繁育目标（Rosvall等，2011）。选育的增长是一个问题，因为它可能导致对实木和纤维尺寸（罗森博格和Cahalan不利影响1997年）。例如，负遗传相关通常年轮宽度之间观察到（如茎直径增长的量度）在挪威云杉（和木材密度或弹性模量（MOE）Chen等人2014年 ; Hannrup等人2004年 ;格兰斯等人2009 ;罗森博格和Cahalan 1997）和其他针叶树（Baltunis等人2007 ; Hong等2014 ; McLean等人2016 ; Wu等人2008）。

生长和实木特性与纤维尺寸的遗传相关性已经受到较少的关注，因为纤维尺寸的确定是耗时和昂贵的。然而，树的生长和纤维维性状间的不利的遗传相关已经在各种针叶树种描述（Hong等2014 ; Lenz等人。2010 ;佐贝尔和杰特1995）。快速增长的树通常具有大的径向纤维宽度（RFW）和切向纤维宽度（TFW），以及低的纤维壁厚（FWT），这表现在低木材密度（Scallan和绿1974年）。例如，Hong et al。（2014年）报道分别RFW和联队，相关木材密度和FWT和较低（0.12 -0.18和）之间的高遗传相关（0.77），在40岁的苏格兰松树（樟子松属）树。在白云杉（云杉青冈（荞）沃斯），早材密度主要受FWT决定的，而晚材的密度主要受RFW决定（伦茨等2010）。

在挪威云杉，生长或实木性能和纤维维度之间的遗传相关都使用两个19岁的克隆试验和一个40岁的全同胞子代试验进行了评估（Hannrup等，2004）。他们发现，密度一般与晚木FWT有显着的正相关，与早木FWT有非显着的遗传相关。然而，木材质量性状的试验的大的变化可以归因于与小群体大小（约200个个体）相关的典型的压制误差。因此，使用更多的实木和纤维性能更彻底的分析是使用涉及纸浆和纸张的加工和产品性能，考虑到个别纤维尺寸可以对纸浆和纸张的不同属性有很大的影响（Kibblewhite更大的数据集需要1999年 ;里德尔等人2005）。一个实例是纤维宽度和壁厚度之间的关系，其是指它们如何容易地塌陷成彼此一致并结合以形成纸张的平带，影响许多性能，例如片密度，强度和表面平滑度威默等人2002）。

针叶树育种的另一个重要方面是减少育种周期，作为加速树木育种过程的一种手段。迄今为止，大多数研究都集中在早期选择增长和实木品质性状，而不是纤维特性（陈等人。2014年，香港等。2015年，库马尔和李2002 ;李和吴2005 ;吴等人2007）。此外，关于年龄相关性和纤维尺寸的早期选择效率的研究不太广泛，尽管它可以为设计纸浆和纸的有效树木育种程序提供基本信息。在挪威云杉现有研究已经报道了纤维长度（佛罗里达州），RFW，TFW，和固定台从髓到树皮表型变异（格尔2004年 ; Lundqvist的等人2002 ; Lundqvist的等人2005 ; Molteberg和Høibø 2007），而年龄-age纤维维度相关和早期选择效率已经估计仅在白云杉（Lenz等人2011），以及在辐射松（固-木材和纤维特性辐射松 D.唐）（Nyakuengama等人1997年），并欧洲赤松（Hannrup和埃克伯格1998年）。

因此，本研究的总体目标是探索基于茎直径和实木木材性质（MOE）的选择对挪威云杉纤维尺寸的影响，以及研究基于RFW，TFW，FWT等的早期选择的可能性。纤维粗糙度（FC）和FL。本研究的详细目的是（1）量化跨越年轮的纤维尺度性状的遗传，（2）探索纤维尺度性状的基因型与环境的相互作用，（3）估计纤维的遗传相关性（4）评估纤维尺度性状的年龄相关性和早期选择效率，以及（5）评价基于选择的结果的潜在影响茎杆直径和纤维尺寸的MOE。我们的研究代表挪威云杉中最广泛的遗传研究，用于年轮的纤维尺寸。
