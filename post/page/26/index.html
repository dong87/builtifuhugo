Porth2013-

Posted on October 1, 2017

Porth I, Klápště J, Skyba O, et al. 2013. Populus trichocarpa cell wall chemistry and ultrastructure trait variation, genetic control and genetic correlations. New Phytologist, 197(3): 777–790. 概要 日益增加的生态和经济的重要性杨树品种和杂交刺激了研究物种的自然变化，并超出其木材质量性状传统的林业活动，以及新兴的生物能源部门遗传控制的程度估计的调查。构建了基于信息，高密度，双重单核苷酸多态性（SNP）遗传标记的实现的亲缘关系矩阵，以估计性状方差分量，遗传力，遗传和表型相关性。 研究了在共同花园图中生长的334 9年龄的杨树毛茛科植物中与木材化学和超微结构相关的17种性状，代表横跨44°至58.6°的纬度范围内的种群。在这些个体中，符合Hardy-Weinberg期望的9342个SNP用于评估基因组双向亲属关系以估计性状之间的狭义遗传和遗传相关性。 所有性状的范围广泛的表型变异是显着的，几个性状遗传度大于0.6。总共鉴定了61个重要的遗传和表型相关性以及高度相关性状的网络。 高特征变异，中度至高遗传的证据，以及鉴定有利特征组合的工业重要特征，有助于为现代工业用途提高杨树育种策略奠定基础。 介绍 杨树种（杨树，白杨和白杨木;整个被称为杨树）是跨越北半球温带和北部地区的重要地理区域的生态和经济上重要的树种（Burns＆Honkala，1990 ; Mansfield＆Weineisen，2007） 。虽然裸子植物主要用于转换成商品，但杨树对于森林产品行业和新的绿色经济来说变得越来越重要，特别是作为提供纸浆和纸张生产的纤维的短轮型物质，定向的刨花板（OSB），模制件和装饰件以及家具部件（Semple 等人，2007）。最近，由于其固有的快速生长速率和有利的细胞壁化学，这些物种已经成为潜在的生物能源作物（Dinus，2000）。在生态效益，能源和二氧化碳平衡方面，木质纤维素原料被认为比基于生物能源生产的基于淀粉和糖的原料更可持续（Yuan et al。，2008）。虽然白杨可以产生相对较高的产量和显着的能量输出（Yuan et al。，2008 ; Carroll＆Somerville，2009），其先天的细胞壁组成需要昂贵的预处理方法来获取复杂的细胞壁碳水化合物（Hendriks＆Zeeman，2009 ; Mansfield ，2009）。已经提出许多因素有助于植物细胞壁解构的这种自然的顽固性（Chang＆Holtzapple，2000 ; Dinus，2000 ; Mosier et al。，2005 ; Studer 等，2011），目前有若干研究策略目前正在努力优化和提高糖化效率（曼斯菲尔德等，1999年，2012年 ; Yuan 等，2008 ; Carroll＆Somerville，2009 ; Shen 等，2011）。 已知生物量积累，木材组成和超微结构性状非常复杂（Dinus，2000 ; Novaes et al。，2009）。还知道杨树在许多重要的木材性质中显示出实质性的变化，例如木质素含量和丁香酰到愈创木酰（S：G）木质素单体的比例（Wegrzyn 等人，2010 ; Studer 等，2011），纤维素和半纤维素组合物（Sannigrahi 等，2010），纤维素结晶度（Kumar 等，2009）和木材密度（Zhang et al。，2012）。杨树改良计划可以利用现有的木材性状的自然变化以及近年来可用于该物种的广泛的基因组资源（Tuskan 等，2006 ; Jansson＆Douglas，2007 ; Ralph et al。 ，2008 ; Geraldes 等人，2011 ; Geraldes 等人，新闻 ; Slavov 等人，2012），以加速改善细胞壁和生长性状的育种策略。已经实施了定量性状位点（QTL）鉴定和其他传统的定量遗传学方法来研究木材形成的复杂遗传结构（Pot et al。，2002 ; Kirst et al。，2004 ; Novaes et al。，2009 ; Thumma et al，2010 ; Wegrzyn et al。，2010 ; Beaulieu et al。，2011）。然而，关于木材性状的遗传控制（遗传力）模式及其遗传关系的知识对于有效的选择育种计划是至关重要的。育种计划的目的是最大限度地发挥工业上重要的木材性状的遗传增益，同时有效地管理不良性状相关性，同时维持未来选择和适应的遗传变异性。为此，需要更好地了解感兴趣的特征之间的继承和关系。除了通常研究的木材密度，通常显示中等至高遗传率（Yanchuk 等人，1984 ; Pliura 等人，2007 ; El-Kassaby 等，2011），对遗传参数知之甚少用于其他与杨木相关的木材性状。本文提供了一个综合评估木材性状，被认为是改善杨树属的实用性的关键。用于工业应用。 [Read More]

Pot2002a-海松-材性-

Posted on October 1, 2017

木材是我们最重要的自然资源之一，已经被利用了数百年的燃料，建筑材料和纸张来源。其组成在物种内和物种之间是变化的。监测特异性变异性的能力是提高木材和最终产品性能的先决条件。本文介绍了一种大型木材性质的遗传控制研究，包​​括生长，木材质量性状，木材化学成分，牛皮纸浆生产参数和纸浆性质，在mari-2时间松木的12×12半透析中Pinus pinaster Ait。）。虽然观察到密度异质性，木质素含量，α-纤维素含量和粗糙度相对较高（hns> 0.3）的狭义遗传，半纤维素，水提取物，牛皮纸浆制备参数和pylodin没有显着的遗传效应。木材密度和纤维性能（长度，宽度，卷曲，零跨度）也获得了较低的遗传力（0.15 <hns <0.3）。因此，考虑到这些性状获得的表型变异系数，通过选择具有优异木材质量的树木进行改良是可行的。然而，似乎很明显，木材质量育种不能在不考虑增长的情况下进行，而管理这种约束的唯一方式（增长与密度之间的负相关）将是精英“木材质量”人口在已经增长中的组成改善遗传种群。

介绍 木材可以被认为是由粘合在一起并由木质素固定的纤维素的柔性管组成的天然复合材料。然而，这种简单的定义隐藏了木材也是非常复杂和可变的，不仅在其化学成分中，而且在解剖学水平（例如管状体形态）。这种变异性不仅发生在物种之间，而且发生在一个物种内，甚至在一棵树内，都是双边的。一方面，木材可以用于各种产品（纸张，建筑材料，化学品，能源等），但另一方面则损害了其在各种应用中的性能。对这种变异性的理解和使用它的能力对于改进最终用途产品至关重要。 海松（Pinus pinaster Ait。）是西南欧重要的商业物种。它是法国的主要针叶树种植面积（140万公顷）和收获产量（每年830万立方米）。其木材用于木材和纸浆行业，涉及不同性状可能感兴趣的不同合作伙伴（森林所有者，木材和纸浆工业）。今天，海上松树育种计划已经实现了第三代的选择。改良品种的遗传增益在体积和直度方面约为30％。木质素选择标准的引入现在被认为是育种计划的重要目标。然而，这种选择受到木材质量特性缺乏信息的阻碍，不仅在遗传上，而且在工业层面。 木材质量只能根据特定的最终用途来定义，并且可能涉及几个性状（例如密度，木质异质性，木材化学成分和纤维性能）。平均密度被认为是木材机械性能的最佳单一预测因子​​[1，36，41，43，48，67]。经常报道弹性模量（MOE），软木的基本力学性能和密度之间的非常强的正相关性（见[54]）。但是，尽管其重要性，平均密度并不是木材力学性能中唯一的特征。 Larson [33]指出，所有木材使用行业面临的最大的木材质量问题缺乏统一性。关于密度，根据Megraw [38]，“比重最大的变异性发生在每个年度环”之内。因此，木材机械性能的育种目标之一可能是各环内密度异质性的降低。如果密度的内环异质性与化学和纤维性质的异质性有关，则此修改也可能影响纸浆生产。假设木材密度与纸浆产量呈正相关（增加密度也增加了每单位面积生产的干纤维的吨位）和一些纸浆质量性状[9,22,25,28]。不幸的是，单独的木材密度是其他牛皮纸浆质量性状的不良指标[28,29]。为了估计树的“纸浆潜力”，重要的是要考虑其纤维特性。最近的研究表明，几种纤维性能可以对纸浆产量和纸浆质量产生显着的影响[16,25,27,68]。也必须考虑化学成分（如木质素，多糖，提取物），这些性状对生产成本和最终产品质量都有直接后果。 为给木材加工业种植树木是一个复杂的问题。确定哪些是影响产品质量的关键属性，（2）估计这些关键性质的遗传改良的可能性，（3）估计其他关键性质选择的相关响应目标性特征如增长。为了提供启动海洋木材质量育种计划的背景，在广泛的实验设计（diallel，因子和克隆试验）中，估计了大量木材性质的遗传参数。本文介绍了一种半透明度中大量木材特性的遗传决定性研究。介绍遗传参数，包括遗传相关性和遗传相关性，并讨论了用于木材和制浆目的的海洋松木利用的育种策略。

R-list-to-data-frame列表转数据框

Posted on October 1, 2017

Assuming your list of lists is called l:

df <- data.frame(matrix(unlist(l), nrow=132, byrow=T))

The above will convert all character columns to factors, to avoid this you can add a parameter to the data.frame() call:

df <- data.frame(matrix(unlist(l), nrow=132, byrow=T),stringsAsFactors=FALSE)

R1--设定工作路径

Posted on October 1, 2017

想要程序正常运行，通常得告诉电脑要执行的程序和文件在哪里，所以第一步是设定工作路径，在R中更是这样。

R有默认的工作路径，执行getwd()，默认工作路径可以在软件中更改。但一般我们的工作项目不止一个，不适合直接在默认路径下操作，而是建立不同的文件夹，然后分别调用。有2种方法： 1. 设定完全工作路径，比如setwd("D:/Sync/R/")； 2. 简化工作路径，比如setwd("./group1")，很明显，默认工作路径或已经设定好的工作路径用一个.表示，这样做也可以在上一级路径中调取这一路径下的文件。

## n-a number or numerical vector
dlm.extract.bb <- function(n) {
for(i in 1:length(n)){
aim<-ev.getven[ev.getven$..count..==n[i],]
aim1<-aim$..values.. 
aim1<-as.data.frame(aim1)
names(aim1)<-aim$..set..
filename<-paste0(substr(names(aim1[1]),2,6),".csv")
write.csv(aim1,filename)
}
print("Job done!")
}

R5--画图-字体设置

Posted on October 1, 2017

函数pdf（或win.metafile）中加入famil="..."可以设置字体。可用的字体​除了”sans”，”mono”，”serif”外还有 ”AvantGarde”, “Bookman”, “Courier”, “Helvetica”, “Helvetica-Narrow”, “NewCenturySchoolbook”, “Palatino” or “Times”。但这个函数的设置是全局性的，如果包含多个分图而分图需要不同字体的话（这个情况极为少见，我是画着玩儿），还是得用par函数，依旧是family="..."的声明。font主要用来更改加粗、斜体等，改变坐标轴标签的方向则是用las函数，比如las=2时，逆时针90°。

#example​

pdf("box.pdf",6,4,family = "NewCenturySchoolbook")
par(mar = c(2, 2,2, .5),mfrow=c(2,2))
boxplot(mpg~cyl,data = mtcars,main="NewCenturySchoolbook")
par(family="sans")
boxplot(mpg~cyl,data = mtcars,main="sans (Helvetica)")
par(family="mono")
boxplot(mpg~cyl,data = mtcars,main="mono (Courier)")
par(family="serif")
boxplot(mpg~cyl,data = mtcars,main="serif (Times)")

R6--报错Error-in-plot-new()---figure-margins-too-large

Posted on October 1, 2017

RStudio中报错Error in plot.new() : figure margins too large。 原因：图的边距太小，就是边距设置的太大。一个是确实小，par("mar")可以查看，可以通过par(mar=c(1,1,1,1))修改。但有时问题依然存在，这是第二个原因，RStudio的四分窗口中plot的窗口过小，解决办法就是用鼠标把窗口拖大点。
