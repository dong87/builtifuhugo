 ## 简介

 什么是breedR R-包执行统计模型专门适用于森林遗传资源分析师。 最终混合模型 ，但不一定容易实现和使用 breedR充当其提供装置在接口 - 结合任意数量的这些模型作为一个较大的组件模型 - 自动计算从几个输入参数， 发病率和协方差矩阵 - 拟合模型 - 绘图数据和结果，并进行模型诊断

 解释变差函数 各向同性变差函数：[\ gamma（h）= \ frac12 V [Z（\ mathbf {u}） - Z（\ mathbf {v}）]，\ quad \ text {dist}（\ mathbf {u}，\ mathbf { v}）= h]

经验各向同性变差是由聚集所有对由（H）分开的点， 不论方向构建。

 解释变差函数 行/列变异函数：[\ gamma（x，y）= \ frac12 V [Z（\ mathbf {u}）-Z（\ mathbf {v}）]，\ quad \ text {dist}（\ mathbf {u} ，\ mathbf {v}）=（x，y）]

实证行/列变差函数是通过聚集所有对正好（X）行（Y）列分隔点的建造。

 解释变差函数 各向异性变异函数：[\ gamma（\ mathbf {x}）= \ frac12 V [Z（\ mathbf {u}） - Z（\ mathbf {v}）] \ quad \ mathbf {u} = \ mathbf { \ pm \ mathbf {x}]

实证各向异性变差函数是通过聚集所有的点对 由分隔同一方向建（| \ mathbf {X} |）。

 竞争

 理论模型 每个人有两个 （未知）育种值（BV）

在direct  BV影响其自身的表型，而competition  BV影响其neghbours“（作为国王的动作）

相邻的效果competition  BVS由它们的和由 （1 / D ^ \阿尔法），其中（\阿尔法） 加权定是称为调谐参数decay

每组BVS的被建模为与由系谱和独立方差 （\西格马^ 2_a）给出的结构矩阵和零均值随机效应 （\西格马^ 2_c）

这两个随机效应与相关联合建模（\ RHO）

 永久环境效应（pec） 环境 （而非遗传）的基础上可选的效果
