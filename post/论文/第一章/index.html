2.2.材料与方法

2.2.1植物材料与试验设计 从初级种子园中选择12个无性系，开展双列交配试验（包含自交）。共获得55个组合，包括8个自交、34对正反交和13个单交（交配设计见表1 - 1）。此外，同年收集12个亲本自由授粉种子。在苗圃中将种子育苗，培育2年后移植到试验林地（1988年）。试验林位于中国辽宁省抚顺市清原县大孤家镇林场（42.72°N，124.88°E），林地海拔约460 m。试验设计采用随机完全区组设计，5个重复，其中重复2和重复4分别包含4个和5个亚重复，密度2 × 2 m。每个组合及自由授粉家系获得的子代个数不完全一致，由于要将全部子代个体布置到林地内，所以小区的大小是变化的，总体上服从每行5株多列(1-3)。试验林四周设有保护行以避免边缘效应。每年进行常规抚育管理。为维持林地的生产力，2009年对试验林进行隔株间伐。 采集12个亲本的新鲜针叶，硅胶干燥，用于后续基因组DNA提取和遗传分析。 表1 - 1全双列交配设计 Table 1 – 1 A full diallel mating design used for the present Japanese larch improvement program

2.2.2数据收集与清洗 分别在2005年和2014年树木休眠期，调查测定试验林内单株的树高（m）、胸径（cm），用H05、H14、DBH05和DBH14表示。树高和胸径分别用测高仪(Vertex Ⅲ, Haglof Company Group, Sweden)和围尺测定。而后利用树高和胸径计算每木的带皮材积指数（m3），用V05、V14表示。材积指数公式为V = H × D2 (1)。 在家系水平和重复水平检查各性状的极端异常值。极端异常值的定义为≤下四分位数减去3倍IQR（interquartile range）的数值，或≥上四分位加上3倍IQR的数值。而后去除极端异常小的数值，因为这些数值的观测个体可能是受压木、遭受严重病虫害或是所处环境条件很差，未能反映真实的生长潜力。 2.2.3 统计分析 因为Yao等(2013)发现使用家系模型分析包含自交的全双列会高估SCA效应，认为不含自交全双列是用于估计GCA和SCA效应的最佳交配设计。因此，在去除自交数据后进行遗传参数估计及遗传效应值预测。混合模型如下： y=Xb+Z_1 gm+Z_2 gd+Z_3 s+Z_4 r+Z_5 p+e 这里，响应变量y是单株表型值列向量。b是固定的重复效应列向量，gm (gd)是随机的亲本GCA效应向量，s是随机的交配组合SCA效应向量，r、p、e分别是随机的正反交效应、小区效应和残差向量。X和Z(Z1-Z5) 分别是将观测值关联到各自固定和随机效应上的设计矩阵。模型（2）中的随机效应假定期望为0并服从正态分布： A亲缘关系矩阵，I是单位矩阵，σ ̂_g^2，σ ̂_s^2，σ ̂_r^2，σ ̂_p^2和σ ̂e^2分别是随机效应gm(gd)，s，r，p，和e的方差估计。 方差估计由限制性最大似然估计（Restricted maximum likelihood，REML）方法获得。各方差的标准误由1阶泰勒展开式决定。利用似然比检验检验（LRT）各方差组分的统计显著性(α = 0.05)。通过解Henderson混合模型方程组(Henderson 1975)求得模型的解，即为包括育种值在内的随机效应值。 为比较各生长性状的相对遗传方差，利用公式(“CV” ) ̂“g” “=” √(“σ” ̂“a” ^“2” +4”σ” ̂“s” ^“2” )/“x” ̅ “×100%” (3)和(“CV” ) ̂“p” “=” √(“σ” ̂“P” ^“2” )/“x” ̅ “×100%” (4)，分别估计了加性遗传变异系数((“CV” ) ̂“g” )和表型变异系数((“CV” ) ̂“p” )。 “σ” ̂“a” ^“2” 和”σ” ̂“s” ^“2” 如上描述，”σ” ̂_“P” ^“2” 是表型方差估计值。x ̅是性状的均值。 狭义遗传力(h2)和广义遗传力(H2)分别按以下2个公式估计： h^2=(4σ_a^2)/(〖2σ〗_a^2+σ_s^2+σ_r^2+σ_p^2+σ_e^2 ) (4) H^2=(4(σ_a^2+σ_s^2))/(〖2σ〗_a^2+σ_s^2+σ_r^2+σ_p^2+σ_e^2 ) (5) 两性状（x、y）遗传相关系数（”r” _“A” ）计算公式为： “r” “A” “=” “σ” ̂(“a” _“x” “a” “y” )/√(“σ” ̂(“a” “x” )^“2” “σ” ̂(“a” _“y” )^“2” ) (7) 以上提及的分析均由ASReml-R 软件包3.0完成(Butler et al. 2009; R. Core Team 2013)。 为调查不同杂交组合个体的生长表现，将来自成对亲本的组合及亲本自交（有的话）和亲本半同胞划分成一组，利用Turkey检验考察组内的差异显著性。 平均杂种优势和自交衰退(SD)的计算公式分别为： Heterosis (%) = (O_i-OP_i)/OP_i ×100% (8)和SD (%) = 〖OP_j-S〗_j/OP_j ×100% (9) O和S 分别代表异交和自交子代的均值。OP是优势亲本半同胞的平均表现，对于自交而言则只利用其亲本半同胞的表现作对照。亲本CF29和Y9没有半同胞子代，以其全部异交子代作为对照。 2.2.4 DNA提取和EST-SSR基因型分型 利用改良的CTAB法(Doyle and Doyle 1987)，提取亲本个体的基因组DNA。 EST-SSR分型的方法及步骤如PCR反应体系、PCR循环程序按照Chen(Chen et al. 2015)的描述进行。共使用了17个EST-SSR标记，引物序列见表2。 利用获得的EST-SSR基因型数据和GenAlEx 6.5软件(Peakall and Smouse 2012)估计亲本群体的遗传多样性指数，如观测等位基因数(Na)、有效等位基因数(Ne)、Shannon 信息指数(I)、观测杂合度(Ho)、无偏期望杂合度(He)。 成对亲本间的Nei遗传距离(Nei 1978)由R软件包poppr Version 2.2.1 (Kamvar and Grunwald 2014)计算。而后利用获得的遗传相似性矩阵及UPGMA（Unweighted Pair Group Method using Arithmetic averages）算法由R软件包Phangorn Version 2.0.4 (Schliep 2011)完成聚类分析。 表1 - 2 17对EST-SSRs引物序列 Table 1 - 2 Details of 17 EST-SSRs used to genotype in L. kaempferi
